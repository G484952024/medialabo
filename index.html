「Webコンテンツ作成」の成果

    本テーマについて

    HTML、CSS、JavaScriptの基本を学び、これらを組み合わせてWebページを作る方法を学ぶ。

    プロジェクト課題について

    プロジェクト課題で作成したページ:プロジェクト課題

    数当てゲーム

    1~10の乱数で決められた数を答えとして,ユーザーは答えを予想して3回まで入力する.

    予想が間違っていたら，予想より答えが大きいか小さいか表示する．

    3回までで答えを当てられたらユーザーの勝ち,そうでなければ負け．

    数当てゲームのページ:課題3-1:数当てゲーム

    各授業回の学習事項

    第1回 WWW と HTML

    HTMLは様々な種類の要素で作られ、その要素はタグで囲まれる。

    p要素は段落、strong要素は強調の働きがある。

    たくさんの種類があるHTMLの要素を使ってWebページの構造を記述していく。

    演習の成果: 演習1-4: HTMLのさまざまな要素

    第２回 CSSによるWebページの装飾

    CSSとはどうゆうものなのかを知り、CSSの種類や調べ方を学習する。

    CSSを使ってWebページのレイアウトや装飾をする。

    レイアウトや装飾の記述の仕方を理解し、レイアウトや色使いを考えて実際にWebページを作る。

    演習の成果: 演習2-3: ウェブブラウザ説明ページの装飾

    第３回 JavaScript の基本

    JavaScriptはWebサイトでボタンを押したり、動きを持たせる役割をしている。

    JavaScriptはコンパイルが必要なく、一行ずつ読み取り実行されるので、コンソールで確認しながらプログラムを書く．

    JavaScriptではエラーが発生しない場合があり、undefinedと表示されることがある。

    演習の成果:演習3-6:オブジェクトの参照

    第４回 DOM と JavaScript

    input要素は主にテキストボックスの作成に必要な要素や属性が記述される。

    button要素はボタンをクリックしたときに次のアクションを起こす時に使われる。

    JavaScriptのプログラムで要素を検索する時は、document.querySelector()で要素を1つずつ、
    document.querySelectorAll()ですべての要素を検索することができる。

    演習の成果:演習4-10:足し算ページ

    第５回 イベント処理

    DOMは要素の作成や要素の編集、削除も行うことができる。

    要素を新しく作成するには、要素のオブジェクトを新規作成し、その要素を入れる場所を決めるためにHTMLの中の要素を検索する。

    検索した要素の直後や内部に新しい要素を置くことができる。

    演習の成果:演習5-7:DOMプログラミング(2)
